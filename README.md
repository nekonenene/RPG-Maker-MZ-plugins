# RPGツクールMZ用プラグイン - RPG Maker MZ Plugins

商用・非商用問わず使用可能な、RPGツクールMZ用のプラグインを置いています。  
クレジット表記には「ハトネコエ」を記載してください。

プラグインの導入方法については、[ツクール公式サイトの講座ページ](https://rpgmakerofficial.com/product/mz/plugin/start/dounyu.html)をご参考に！  
ダウンロードした `HTN_xxx.js` のような名前のファイルを、プロジェクト内の `js/plugins` フォルダーの中に入れてください。

## 🦘 HTN_EnemyJumpOnTurn

敵が攻撃ターンで軽くジャンプするように動き、誰の攻撃かをわかりやすくするプラグインです。

### 詳しい使い方はこちら：

**https://github.com/nekonenene/RPG-Maker-MZ-plugins/tree/main/my_plugins/HTN_EnemyJumpOnTurn**

MP回復やTP回復の反転にも対応しています。

**https://github.com/nekonenene/RPG-Maker-MZ-plugins/tree/main/my_plugins/HTN_ZombieState**
